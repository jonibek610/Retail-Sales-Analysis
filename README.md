# Retail-Sales-Analysis
This project analyzes a retail dataset to identify key sales drivers, customer demographics, and seasonal trends. The goal was to provide actionable recommendations for inventory management and targeted marketing.

.

# Source : Retail Sales Dataset(Kaggle)
Features: Has columns like Transaction IDs, Date, Customer ID, Gender, Age, Product Category, Quantity, and Price

link to the original dataset used : https://www.kaggle.com/datasets/mohammadtalib786/retail-sales-dataset

# Key Findings
Electronics is the highest sales category with its major peak in May.

Average customer is 41 y.r old. Female customers slightly purchase more than males.

Sales significantly dip in September across all categories, identifying a clear opportunity for "Back-to-School" promotional campaigns.
